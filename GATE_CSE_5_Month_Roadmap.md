# GATE CSE 2027 — 5 Month Preparation Roadmap

**Based on:** GATE CS Syllabus (IIT Guwahati, GATE 2026) + Subject-wise Weightage

## Overall Structure (5 Months = ~20 Weeks)

| Phase | Duration | Focus |
|---|---|---|
| Phase 1 | Month 1 (Week 1–4) | Foundation subjects — PDS, Discrete Math, Digital Logic |
| Phase 2 | Month 2 (Week 5–8) | Engineering Mathematics, Algorithms, COA |
| Phase 3 | Month 3 (Week 9–12) | Operating System, Databases, Theory of Computation |
| Phase 4 | Month 4 (Week 13–16) | Computer Networks, Compiler Design, GA + Revision start |
| Phase 5 | Month 5 (Week 17–20) | Full Revision, PYQs, Mock Tests, Weak Areas |

**Daily routine suggestion:** 6–7 hours/day (working/college students: 4–5 hrs) — 60% new topic, 25% practice/PYQs, 15% short notes/revision.

---

## MONTH 1: Foundation (Week 1–4)

### Week 1–2: Programming and Data Structures
- Programming in C — syntax, pointers, memory, recursion (deeply — trace recursion trees, master it, GATE loves recursion output questions)
- Arrays, Stacks, Queues, Linked Lists (singly, doubly, circular)
- Practice: Trace-based questions daily (write code, predict output)

### Week 3: Data Structures (Advanced)
- Trees — Binary Trees, Binary Search Trees (insertion, deletion, traversal complexity)
- Binary Heaps — heapify, build-heap complexity
- Graphs — representation (adjacency list/matrix)
- Practice: 25–30 PYQs from Programming + DS

### Week 4: Discrete Mathematics
- Propositional & First Order Logic
- Sets, Relations, Functions, Partial Orders, Lattices
- Monoids, Groups
- Graph Theory — connectivity, matching, colouring
- Combinatorics — counting, recurrence relations, generating functions
- Practice: 20 PYQs

**Weekend tasks (every week):** 1 short revision note (1 page) per topic + solve last 5 years' PYQs of that topic.

---

## MONTH 2: Math + Core CS (Week 5–8)

### Week 5: Digital Logic
- Boolean Algebra, K-Map minimization
- Combinational Circuits (adders, MUX, decoders)
- Sequential Circuits (flip-flops, counters, registers)
- Number representation & computer arithmetic (fixed & floating point — IEEE 754 is high-yield)

### Week 6: Engineering Mathematics — Part 1
- Linear Algebra: Matrices, determinants, system of equations, eigenvalues/eigenvectors, LU decomposition
- Calculus: Limits, continuity, differentiability, maxima-minima, mean value theorem, integration

### Week 7: Engineering Mathematics — Part 2
- Probability & Statistics: Random variables, distributions (uniform, normal, exponential, Poisson, binomial), mean/median/mode/SD, conditional probability, Bayes theorem
- Practice: 30 PYQs (Engg Maths is scoring — don't skip)

### Week 8: Algorithms
- Searching, Sorting, Hashing — complexity analysis of each
- Asymptotic notations (worst case time/space)
- Design techniques: Greedy, DP, Divide & Conquer (solve 15+ DP problems)
- Graph algorithms: BFS/DFS, MST (Prim's/Kruskal's), Shortest paths (Dijkstra, Bellman-Ford, Floyd-Warshall)

---

## MONTH 3: Systems (Week 9–12)

### Week 9: Computer Organization and Architecture
- Machine instructions, addressing modes
- ALU, datapath, control unit
- Instruction pipelining, pipeline hazards (numerical-heavy — practice a lot)
- Memory hierarchy: cache (mapping techniques, hit/miss ratio numericals), main memory, secondary storage
- I/O interface: interrupt, DMA

### Week 10–11: Operating System
- System calls, processes, threads, IPC
- Concurrency & synchronization (semaphores, monitors — classic problems: producer-consumer, dining philosophers)
- Deadlock (detection, avoidance — Banker's algorithm)
- CPU scheduling (FCFS, SJF, Round Robin, Priority — numericals)
- Memory management & virtual memory (paging, segmentation, page replacement algorithms — numericals)
- File systems

### Week 12: Theory of Computation
- Regular expressions, Finite Automata (DFA/NFA construction & minimization)
- Context-free grammars, Push-down automata
- Regular/CFL properties, Pumping lemma
- Turing machines, Undecidability
- Practice: 20–25 PYQs

---

## MONTH 4: Applied CS + GA (Week 13–16)

### Week 13: Databases
- ER model, Relational model (relational algebra, tuple calculus)
- SQL (write queries — very scoring)
- Integrity constraints, Normal forms (1NF–BCNF — practice normalization problems)
- File organization, Indexing (B, B+ trees)
- Transactions & concurrency control

### Week 14–15: Computer Networks
- Layering: OSI, TCP/IP
- Switching: packet, circuit, virtual circuit
- Data link layer: framing, error detection, MAC, Ethernet bridging
- Routing protocols: shortest path, flooding, distance vector, link state
- IP addressing: IPv4, CIDR, fragmentation, ARP/DHCP/ICMP, NAT
- Transport layer: flow/congestion control, UDP, TCP, sockets
- Application layer: DNS, SMTP, HTTP, FTP, Email

### Week 16: Compiler Design + General Aptitude
- Lexical analysis, parsing, syntax-directed translation
- Runtime environments, intermediate code generation
- Data flow analysis: constant propagation, liveness, common subexpression elimination
- GA: Start dedicated practice (verbal, numerical ability, quant reasoning) — 30 min daily from here onward

**➡️ Syllabus complete by end of Week 16.**

---

## MONTH 5: Revision + Mock Tests (Week 17–20)

### Week 17: Subject-wise Rapid Revision (Part 1)
- Day 1: Discrete Math + Engg Math
- Day 2: Digital Logic + COA
- Day 3: PDS + Algorithms
- Day 4: Full-length Mock Test 1 + Analysis (spend 1 full day only analyzing mistakes)
- Day 5–7: OS + DB revision + 1 sectional test each

### Week 18: Subject-wise Rapid Revision (Part 2)
- Day 1–2: TOC + Compiler Design
- Day 3–4: Computer Networks
- Day 5: Full-length Mock Test 2 + Analysis
- Day 6–7: GA + weak topics from Mock 1 & 2

### Week 19: Previous Year Questions (PYQs) — Last 10 Years
- Solve GATE CS PYQs (2015–2025) topic-wise, timed
- 2 full-length mocks this week (Mock 3, Mock 4) with detailed error log

### Week 20: Final Revision + Test Series
- Revise all short notes (1-pagers made throughout journey)
- 2–3 full mock tests under exam conditions (same time slot as GATE)
- Focus only on weak areas identified from error log
- No new topics — only consolidation

---

## Key Strategy Tips

1. **Make short notes** — 1 page per topic, formulas + tricky points. This is your final revision material.
2. **Error log** — maintain a spreadsheet: mistake type (conceptual/silly/time), topic, mock number. Review weekly.
3. **PYQs are gold** — solve at least last 15 years' papers topic-wise, not just full mocks.
4. **Numerical-heavy subjects** (COA, OS, CN, Probability) need daily practice, not just reading.
5. **GA** — don't ignore; 15 marks is significant. Practice 15–20 min daily from Month 2 onward itself.
6. **Mock test frequency** in last month: minimum 6–8 full-length mocks under real exam conditions.
7. **Weekly self-test**: every Sunday, take a mixed-topic quiz of everything studied that week.
